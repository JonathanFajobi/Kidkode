Documentation

This application was created with the intention of helping the users learn as fast as possible. To make use of the app correctly please read the full documentation below.


First of all, whenever answering questions make sure to answer the multiple choice questions, before the IDE quesiton. You can use it to work out or try solutions.
But remember - always work out the multiple choice question first.
If you need help during an exercise, click the need help button. This should make it a little bit easier for you.


Try to tackle stages and lessons in a chronological order. This will ensure that you have the prior knowledge required.


How to use this application
  • Make sure to read pop ups - these contain the most useful information
  • You can utilise the code editor on the right hand side to test out mulitple choice answers
  • Use arrow buttons to go to the next or previous page
